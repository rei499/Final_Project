# Human Error: 404

### **Project Overview**

_Human Error: 404_ is a text-based choice-driven game developed as a final project for the course Intermediate Programming. It's a simple student life simulator, exploring the systemic and social pressures faced by students in the Philippines, through two distinct narrative paths: **Survival** and **Loyalty**.

### **How to Play**

1. **Launch the Game**: Run the script in any Python-compatible terminal (e.g., VS Code or Command Prompt)
2. **Choosing your Path**:
   - **Path 1**: Navigate financial hardships and dilemma of illegal labor vs. one's integrity.
   - **Path 2**: Navigate social pressure and the dilemma of academic responsibility vs. peer loyalty.
